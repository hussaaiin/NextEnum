# NextEnum

NextEnum is a Python command-line tool that helps cybersecurity students and junior pentesters move from Nmap scan results to a clear enumeration plan.

The goal is not to automate exploitation or replace thinking. NextEnum is designed to act as a terminal-based enumeration guide for authorized labs and CTFs.

## Planned Features

- Parse Nmap normal text and XML output
- Show detected services clearly
- Recommend enumeration order
- Explain why each service should be prioritized
- Provide service specific enumeration walkthroughs
- Track commands used during enumeration
- Export clean plain text notes

## Project Status

This project is currently in early development.
